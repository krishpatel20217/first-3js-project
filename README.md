# React Dog 🐶

A 3D dog model viewer built with React, Vite, React Three Fiber, and Drei.

## Features

- 3D GLB dog model
- React Three Fiber rendering
- Orbit controls
- Fast development with Vite

## Tech Stack

- React
- Vite
- Three.js
- @react-three/fiber
- @react-three/drei

## Installation

```bash
git clone https://github.com/YourUsername/react-dog.git
cd react-dog
npm install
npm run dev
